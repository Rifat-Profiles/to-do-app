## Features

- Daily task list with priority levels and reminders
- 7-day default weekly schedule builder
- Drama and anime airing date tracker
- Habit builder and bad habit eliminator
- Progress dashboard with streaks and charts
- App lock via fingerprint, face ID, or custom PIN
- Focus timer for deep work sessions
- 100% local storage — no account, no cloud, no tracking
